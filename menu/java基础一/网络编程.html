# 网络编程

  1. IP地址 设备在网络中的地址，是唯一的标识
  2. 端口 应用程序在设备中的唯一协议
  3. 协议 数据在网络中传输的规则，常见的协议有UDP协议和TCP协议

# IP

分配给上网设备的数字标签

  • ipv4 32bit（4进制） 点分十进制表示法

  • ipv6 128bit（16进制）冒分十六进制表示法

  • 方法

    • ipconfig 查看本机ip
    • ping IP地址 检查与此ip是否可连接

  • Inetaddress

    • getByName(主机名 | ip)
    • getHostName 获取主机名
    • getHostAddress 获取ip

# 端口

取值范围 0 - 65535 建议使用 1024以上的端口

# 协议

  • UDP 面向无连接通信协议 速度快，有大小限制一次最大发送64K，数据不安全，易丢失数据
  • TCP 传输控制协议 面向连接通信协议
    • 速度慢，没有大写限制，比较安全

# UDP通信程序

    /* 发送端 */
    // 创建 DatagramSocket 对象
    DatagramSocket ds = new DatagramSocket();
    // 发送数据容器 地址 端口
    String s = "这是数据";
    byte[] bytes = s.getBytes();
    InetAddress address = new InetAddress.getByname("127.0.0.1");
    int prot = 3300;
    DatagramPacket dp = new DatagramPacket(
        bytes,
        bytes.length,
        address,
        prot
    );

    ds.send(dp);

    ds.close();

    /* 接收端 */
    // 创建 DatagramSocket 对象 形参需要与发送端的端口保持一致
    DatagramSocket ds = new DatagramSocket(3300);

    // 获取数据容器
    byte[] bytes = new byte[1024];
    DatagramPacket dp = new DatagramPacket(
        bytes,
        bytes.length
    );

    // 把数据放入容器中
    ds.receive(dp);

    // 从容器中获取数据
    byte[] data = dp.getData();
    int lengh = ds.getLengthb(); // 获取字节长度
    ds.close();

    /* 客户端 */
    // 1. 创建Socket对象
    Socket socket = new Socket("127.0.0.1", 3300);
    // 2. 获取一个IO流开始写数据
    OutputStream os = socket.getOutputStream();
    os.write("hello".getBytes());

    // 关闭输出流，结束标记
    socket.shutdownOutput();

    // 获取服务器响应
    // InputStream is = socket.getInputStream();

    // 字符流
    InputStream is = socket.getInputStream();
    BufferedReader br = new BufferedReader(is);
    String line;
    // read 也是阻塞的
    while((line = br.readLine()) != null) {
        
    }
    // 3. 释放资源
    os.close();
    is.close()
    br.close();
    socket.close();

    /* 服务器 （一般服务器是不关闭的）
        使用 线程进行优化
    ） */
    // 1. 创建Socket对象
    ServerSocket ss = new ServerSocket(3300);
    // 2. 监听客户端连接 阻塞 作用是等待客户端连接
    Socket accept = ss.accept();
    // 3. 获取一个IO流开始读数据 （ 字节流 | 字符流 ）
    InputStream is = new accept.getInputStream();
    int b;
    // read 也是阻塞的
    while((b = is.read()) != -1) {
        (char) b;
    }
    OutputStream os = socket.getOutputStream();
    os.write("你谁啊".getBytes());
    // 4. 释放资源
    is.close();
    os.close();
    ss.close();
