# File

文件操作

  • 构造
    • new File(全路径);
    • new File(父路径, 子路径);
    • new File(new File(路径), 子路径);
  • 路径
    • 绝对路径 从盘符开始
    • 相对路径 相对于项目而言
  • 方法
    • createNewFile 创建文件 创建成功true 不成功false
    • mkdir（被mkdirs替代） 创建文件夹 单级创建 创建成功true 不成功false
    • mkdirs（推荐使用） 创建文件夹 多级创建 创建成功true 不成功false
    • delete 删除文件（空文件夹 有内容先删除内容） 删除成功true 不成功false （不能通过回收站还原）
    • isFile 判断是否为文件
    • isDirectory 判断是否为文件夹
    • exists 判断此抽象路径的文件是否存在
    • getName 调用者是文件那么获得的是文件名和后缀名 调用者是文件夹那么获得的是文件夹名字
  • 高级获取功能
    • listFiles 进入文件夹 获取文件与文件夹的file类型的数组（包括隐藏文件或者文件夹） 调用者是一个文件时或者需要权限的文件夹时返回是null

# 字节流

处理非文本文件

  • 方法
    • FileOutputStream
      • 示例（写）
        • 参数1：路径，参数2：打开续写功能
            // 1. 创建子节输出流
    // 文件如果存在 会进行覆盖
    FileOutputStream fos = new FileOutputStream(路径);
    // 2. 写数据 一次一次写
    fos.write(97);
    // 3. 释放资源
    fos.close();

    <!-- 写 -->
    // 一次写一个字节数组
    byte [] bys1 = {1,2,3};
    fos.write(bys1);

    // 一次写一个字节数组一部分
    byte [] bys2 = {1,2,3,4,5,6,7,8};
    fos.write(bys2, off: 1, len: 2)

    <!-- 换行 -->
    // windows \r\n
    // linux \n
    // mac \r
    fos.write(97);
    fos.write("\r\n".getBytes());
    • FileinputStream
      • 示例（读）
        • 一次读一个
            // 1. 创建子节输入流
    FileinputStream fis = new FileinputStream(路径);
    // 2. 读数据 一次一次读
    int read = fis.read();
    // 3. 释放资源
    fis.close();

    <!-- 读数据 -->
    int i;
    // 等于-1 表示没内容 退出循环
    while((i = fis.read()) != -1) {
        // 输出内容
    }
      • 一次读多个 && 一次写多个
          // 1. 创建子节输出流
    FileinputStream fis = new FileinputStream(路径);
    FileOutputStream fos = new FileOutputStream(路径);

    // 创建一个字节数组 （相当于一个篮子 一次运多少）
    byte [] bytes = new byte[1024];
    int len; // 本次读到的有效字节个数，这次读了几个字节
    while((len = fis.read(bytes) != -1)) {
        fos.write(bytes, 0, len);
    }

    fis.close();
    fos.close();

# 字节缓冲流 （推荐使用）

提高字节读取速度 ( 类似于将货物放在一个容器中，盛满了在处理 )

  • BufferOutputStream 字节缓冲输出流
  • BufferedInputStream 字节缓冲输出流
  • 示例
    // 在底层创建了 8192 长度的字节
    BufferedInputStream bis = new BufferedInputStream(
        new FileinputStream(路径)
    )
    // 在底层创建了 8192 长度的字节
    BufferOutputStream bos = new BufferOutputStream(
        new FileOutputStream(路径)
    )

    // 每次倒手一个字节
    int len;
    while((len = bis.read(bytes) != -1)) {
        bos.write(len);
    }

    bis.close();
    bos.close();

    /* 结合数组使用 */
    // 每次倒手多个字节
    byte [] bytes = new byte[1024];
    int len1;
        while((len1 = bis.read(bytes) != -1)) {
        bos.write(bytes, 0, len1);
    }

# 字符流

  • 编码表
    • 特点
      • 存 转二级制 编码
      • 读 二进制解析 解码
      • 编码与解码必须一致 否则会乱码
      • 中文的编码 都是 负数开头
    • 编码表
      • ASCII 没有中文
      • GBK 有中文 兼容 ASCII 一个中文两个子节
      • Unicode 万国码 以UTF-8（编码格式）编码后 一个中文以三个字节形式存储 （主要使用）
    • 编码
      • byte[] getBytes() 使用平台字符集 将String编码为一系列字节
      • byte[] getBytes(String charsetName) 使用指定字符集 将String编码为一系列字节
          String s1 = "你好";
    byte[] bytes1 = s1.getBytes();

    String s2 = "你好";
    byte[] bytes2 = s2.getBytes("UTF-8");

    String s3 = "你好";
    byte[] bytes3 = s3.getBytes("GBK");
    • 解码
      • String(byte[] bytes) 使用平台的默认字符集解码指定的字节数组来构造新的String
      • String(byte[] bytes, String, charsetName) 使用指定的字符集解码指定的字节数组来构造新的String
          byte[] bytes1 = {22, 33, 55, 66};
    String s1 = new String(bytes1);
    String s2 = new String(bytes1, "gbk");
    String s3 = new String(bytes1, "UTF-8");
    • 乱码原因
  • FileWriter 字符输出流
    • flush 刷新流
    • write
        // 方式1
    FileWriter fw = new FileWriter(
        new File(路径)
    );
    // 方式2 推荐使用
    FileWriter fw = new FileWriter(路径);
    // 一次写一个字符
    fw.write(97);

    // 一次写一个字符数组
    char[] chars = {97, 98, 100};
    fw.write(chars);

    // 一次写一个字符数组的一部分
    char[] chars = {97, 98, 100};
    fw.write(chars, 0, 1);

    // 一次写一个字符串
    String str = "你好";
    fw.write(str);

    // 一次写一个字符串的一部分
    String str = "你好";
    fw.write(str, 0, 1);

    fw.close();
  • FileReader 字符输入流
        // 方式1
    FileReader fr = new FileReader(
        new File(路径)
    );
    // 方式2 推荐使用
    FileReader fr = new FileReader(路径);

    // JDK11后 字符流推出一个构造，可以解决指定编码表(不必使用转换流处理乱码问题)
    FileReader fr = new FileReader(路径, Charset.forName("gbk"));

    // 一次读取一个字符
    int ch;
    while((ch = fr.read()) != -1) {
        (char) ch;
    }

    // 一次读取多个字符 chars 就是容器 fr.read(chars) 放入到容器的操作
    char[] chars = new char[1024];
    int len;
    while((len = fr.read(chars)) != -1) {
        new String(chars, 0, len);
    }

    fr.close();

# 转换流

  • 输入 InputStreamReader 字节流到字符流的桥梁
  • 输出 OutputStreamWriter 字符流到字节流的桥梁
  • 使用场景
        // 读文件, 解决乱码 
    InputStreamReader isr = new InputStreamReader(
        new FileinputStream(路径),
        "gbk"
    );

    int ch;
    while((ch = isr.read()) != -1) {
        (char) ch;
    }

    isr.close();
    
    // 使用指定编码形式写文件
    OutputStreamWriter osw = new OutputStreamWriter(
        new FileOutputStream(路径),
        "gbk"
    );
    osw.write("我爱学习");
    osw.close();

# 特殊操作流

  • 对象操作流

    • 输出流 （序列化流） ObjectOutputStream
          // User 类需要实现一个接口 Serializable（标记接口，表示这个类可以被序列化）
    User user = new User("zhangsan", "qwer");

    ObjectOutputStream oos = new ObjectOutputStream(
        new FileOutputStream(路径)
    );

    // 以字节形式将对象写入本地
    oos。writeObject(user);

    oos.close();
    • 输入流（反序列化流） ObjectInputStream
          ObjectInputStream ois = new ObjectInputStream(
        new FileinputStream(路径)
    );
    while(true) {
        try{
            User o = (User) ois.readObject();
        }catch(EOFException e) {
            break;
        }
    };

    // 或者 Arraylist 直接 for循环 取出
    ois.close();
    • 序列号报错
      • javabean 修改后 序列化的序列号 与 反序列化序列号 对不上 所以报错
      • 解决问题 给对象所属类加一个serialVersionUID private static final long serialVersionUID = 22YY;
      • transient 不让属性参与序列化过程

  • Properties

    • 是一个Map体系集合类
    • Properties有跟IO相关的方法
    • 键值对的数据类型基本都定义为字符串
    • 特有属性
      • setProperty (与map的put相同)
      • getProperty (与map的get相同)
      • stringProperty (与map的KeySet相同)
    • 与IO流相关的方法
      • 将本地文件中的键值对数据读取到集合中
            Properties prop = new Properties();
    FileReader fr = new FileReader(路径);
    prop.load(fr);
    fr.close();
      • 将集合中的数据以键值对形式保存在本地
            Properties prop = new Properties();
    prop.put("a", "1");
    prop.put("b", "2");
    FileWriter fw = new FileWriter(路径);
    prop.store(fw, null); // 参数二为注释（没需求的话直接给null）
    fw.close();
